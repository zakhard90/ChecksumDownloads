ChecksumDownloads

Lightweight Windows utility that automatically computes and displays SHA-256 checksums for newly downloaded installers and archive files.

What it does

ChecksumDownloads runs quietly in the background, watches download locations, and displays SHA-256 hashes via a Windows toolbar notification when new files appear.

Key features

  • Watches common download locations automatically
  • Computes SHA-256 for installers and archives
  • Displays the checksum in a toolbar notification (toast)
  • Runs silently in the background
